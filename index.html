研究项目

AMAR Engine - 3D虚拟世界生成工具

AMAR Engine (AME) 是一款下一代3D工具，旨在从自然语言描述和现实世界捕获中生成虚拟世界，结合了人工智能和计算机图形学技术。

GitHub Clone: git@github.com:Oasis-Company/AMAR-Engine.git

Kethematics - 家庭研究新领域

Kethematics 是一个新提出的研究领域，探索家庭在人类历史中的作用及其对个人、社会和文化的深远影响，结合了认知科学和数学方法。

GitHub Clone: git@github.com:Oasis-Institute-of-Technology-Science/Kethematics.git

LinOasis - 线性代数可视化工具

LinOasis 是一个交互式Web应用程序，提供线性代数和矩阵运算的可视化和教育资源，帮助学生和研究人员理解复杂的数学概念。

访问应用 访问地址: /linoasis
